# ZL Portfolio

Interactive dashboard for the ZL (Junior-1) multi-asset portfolio — a 10-fund mix across global equities, alternatives, bonds, and cash, built for a non-expert audience with plain-language explanations throughout.

## Contents
- **10-year backtest** (Jun 2016 – Jun 2026): growth curve vs benchmarks, drawdowns, calendar-year returns
- **10-year Monte Carlo projection** under high political/global uncertainty (5 weighted scenarios)
- **Allocation** by sleeve and by fund, plus currency (FX) exposure
- **Risk vs reward** bubble chart and **correlation heatmap**
- **Full holdings table** (sortable) and **plain-language notes** on each choice
- **Glossary** of all financial terms

## Tech
Static single-page site: `index.html`, `style.css`, `app.js`, and embedded data in `data.js`. Charts via Chart.js.

## Disclaimer
Research and analysis only, not personalized financial advice. Backtests and projections do not guarantee future results. Currency basis: THB.
